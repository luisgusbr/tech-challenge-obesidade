# Sistema Preditivo de Obesidade
🎯 Previsão do nível de obesidade utilizando Machine Learning
📊 Dashboard analítico integrado
🌐 Aplicação desenvolvida em Streamlit
🤖 Modelo Random Forest treinado com Feature Engineering

# Sobre o Projeto
Este projeto foi desenvolvido como parte do Tech Challenge da Pós-Tech FIAP, com o objetivo de construir uma solução de Ciência de Dados capaz de prever o nível de obesidade de um paciente a partir de características físicas, hábitos alimentares e estilo de vida.
A solução contempla todo o ciclo de Machine Learning, desde a preparação dos dados até a disponibilização do modelo em produção através de uma aplicação web desenvolvida em Streamlit.
Além da previsão, foi construída uma visão analítica contendo indicadores e gráficos para apoiar equipes médicas na identificação de padrões relacionados à obesidade.
